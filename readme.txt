This mini-project is made by Rémy Gaudré

Here is some explenation of the folders:

- LocalDev_Env: the main project repository, you can develop the WebApp here in local environment. It provide also a VM to deploy locally the Web application and a test folder.

- Pipeline_Env: this folder is used to host gitlab and the main gitlab-runner environment.

- Stage_sHWE_Env: this folder is used to create a stage vm and permit to make some test before deploy in production environment.

- Prod_sHWE_Env: this folder is used to create a production VM.
